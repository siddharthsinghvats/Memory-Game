Refer to rules here.

  • This is a memory game.
  • You need to remember the sequence in which the color flash on the screen.
  • You need to click on the colors in the same sequence as they appeared
  • Sequence should be selected from start at each step.
  • Example: If red flashed first then click red, then if blue flashed click on red then blue,then if yellow appeared choose red,then blue and then yellow.
  • You should choose the sequence from start at each step.
  • Have a great game!!
